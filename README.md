## Current Project

Atualmente estou desenvolvendo uma plataforma própria de **Threat Intelligence**, baseada na criação e no enriquecimento de uma base de **Indicators of Compromise (IoCs)**.

O objetivo é consolidar indicadores provenientes de diversas fontes, normalizá-los, enriquecê-los com contexto e disponibilizá-los para correlação, detecção e resposta a incidentes.

### Principais funcionalidades

* Coleta automatizada de IoCs
* Normalização e deduplicação
* Enriquecimento de indicadores
* Correlação entre eventos
* Confidence Scoring
* APIs para integração
* Exportação em múltiplos formatos
* Integração com SIEM, XDR e SOAR
